## 🔌 Electric Circuits II Lab Project
**Course:** ELCT 401 – Electric Circuits II  
**Languages/Tools:** 🎛️ PSpice, 📊 Oscilloscope, 🔧 Breadboard  

📖 **Description (from PDF):**  
The project required designing **band-pass** and **band-stop filters** to meet specific frequency requirements. The task included theoretical design, simulation in PSpice, and hardware implementation. Students had to calculate cutoff frequencies, verify gain and bandwidth, and compare theoretical vs experimental results.  

✅ **Outcome:** A fully working filter circuit that matched theory, simulation, and hardware results.  
📄 [Read Full Report (PDF)](https://github.com/FerialAbdulrahem/Semester4/blob/main/S4Projects/Circuits/Project%20report.pdf)

## 💻 Computer Organization Project
**Course:** CSIS 402 – Computer Organization  
**Languages/Tools:** 🖥️ Logisim  

📖 **Description (from PDF):**  
The project aimed to build a **basic computer architecture** in Logisim, then extend it into a programmable system with a control unit. Milestone 1 focused on registers, memory, and ALU design. Milestone 2 added a control unit, instruction cycle, and execution of assembly instructions.  

✅ **Outcome:** A complete Logisim computer capable of running assembly programs with arithmetic, logic, branching, and looping.  
📄 [Read Full Description (PDF)](https://github.com/FerialAbdulrahem/Semester4/blob/main/S4Projects/CSIS%20402/Milestone%202/Project%20milestone%202%20.pdf)

## 🎶 Signals & Systems – Audio Processing
**Course:** COMM 401 – Signal and System Theory  
**Languages/Tools:**  Python (NumPy, Matplotlib, SciPy, SoundDevice)  

📖 **Description (from PDF):**  
The project involved generating piano signals, adding noise, and removing it using FFT. Students had to demonstrate signal synthesis, noise modeling, and frequency-domain filtering. Deliverables included plots, audio playback, and analysis of clean vs noisy signals.  

✅ **Outcome:** A clean audio signal restored after noise cancellation, with clear plots and audible improvement.  
📄 [Read Full Description (PDF)](https://github.com/FerialAbdulrahem/Semester4/blob/main/S4Projects/Audio%20Processing/Signals%20Project.pdf)

## 🎲 Jackaroo Game – Programming Lab
**Course:** Programming Lab  
**Languages/Tools:** ☕ Java (JavaFX GUI), 💻 C++ (logic prototyping)  

📖 **Description (from PDF):**  
The project was to implement a digital version of the **Jackaroo board game**. Requirements included card deck design, marble mechanics, CPU opponent logic, and a graphical interface. The game had to support single-player mode against CPUs with strategic rules.  

✅ **Outcome:** A fully playable Jackaroo adaptation with custom rules, CPU opponents, and a JavaFX-based interface.  
📄 [Read Full Description (PDF)](https://github.com/FerialAbdulrahem/Semester4/blob/main/S4Projects/Jackroo/Game%20Description.pdf)

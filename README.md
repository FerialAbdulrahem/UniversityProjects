# 🎓 Semester 4 Projects – German University in Cairo

This repository contains all my **Semester 4 projects**. Each project demonstrates a different skill set — from hardware circuits to digital computer design, signal processing, and game development.  

---

## 📂 Project Overview

| 🌟 Project | 📘 Course | 🛠️ Languages/Tools | 📖 Description | ✅ Outcome |
|------------|-----------|--------------------|----------------|------------|
| 🔌 **Electric Circuits II Lab Project** | ELCT 401 | 🎛️ PSpice, 📊 Oscilloscope, 🔧 Breadboard | Designed and implemented **band-pass** and **band-stop filters**. Started with theoretical equations, simulated in PSpice (AC sweep & time domain), then built hardware on a breadboard using LM741 op-amps. Verified performance with oscilloscope readings at f < fo, f = fo, f > fo. | ⚡ A fully working filter circuit that matched theory, simulation, and hardware results. |
| 💻 **Computer Organization Project** | CSEN/CSIS 402 | 🖥️ Logisim | Built a **digital computer architecture** with memory, registers, and ALU. Added a control unit with decoders, counters, and logic gates. Implemented instructions (`LDA, MUL, ADD, STA, ISZ, BUN, AND`) and wrote RTL micro-operations. Verified execution cycle-by-cycle with timing diagrams. | 🏗️ A complete Logisim computer capable of running assembly programs with arithmetic, logic, branching, and looping. |
| 🎶 **Signals & Systems – Audio Processing** | COMM 401 |  Python (NumPy, Matplotlib, SciPy, SoundDevice) | Generated piano note signals from 3rd & 4th octaves, combined them into a melody, added random sinusoidal noise, then removed it using **FFT filtering**. Produced time-domain and frequency-domain plots and confirmed audio playback quality. | 🎵 A clean audio signal restored after noise cancellation, with clear plots and audible improvement. |
| 🎲 **Jackaroo Game – Programming Lab** | Programming Lab | ☕ Java (JavaFX GUI), 💻 C++ (logic prototyping) | Reimagined **Jackaroo board game** as a single-player version vs 3 CPUs. Designed a 102-card deck with custom rules. Implemented marble mechanics (movement, swapping, burning, saving, traps, safe zones, firepit). Programmed CPU logic for strategic play and built a JavaFX GUI to visualize the board and actions. | 🎮 A fully playable Jackaroo adaptation with custom rules, CPU opponents, and a JavaFX-based interface. |
